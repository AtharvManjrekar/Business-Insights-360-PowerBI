## Project Overview
Business Insights 360 is an end-to-end Power BI dashboard designed to provide a unified view of business performance across Finance, Sales, Marketing, Supply Chain, and Executive functions.

## Problem Statement
Organizations often rely on siloed reports across departments, making it difficult to gain a consolidated view of performance and make timely decisions.

## Solution
Built a centralized Power BI solution with five interconnected business views, enabling stakeholders to track KPIs, profitability, operational performance, and business trends from a single source of truth.

## Tech Stack
- Power BI
- DAX
- Power Query
- SQL

## Key Features
- Dynamic P&L Statement
- Top N Analysis (RANKX)
- Bookmark Navigation
- Executive KPI Dashboard
- Profitability & Risk Analysis
- Interactive Filtering

## Dashboard Views
1. Executive View
2. Finance View
3. Sales View
4. Marketing View
5. Supply Chain View
